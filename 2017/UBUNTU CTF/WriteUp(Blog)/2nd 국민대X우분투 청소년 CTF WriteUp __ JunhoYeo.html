2nd 국민대X우분투 청소년 CTF WriteUp

2017.12.10 19:53

"우부우부 우분투"

2017.11.05

김현준, 홍다오라는 친구들과 함께 우분투 CTF에 신청했다. 

팀 이름은 '현준다방'. KAIST 사이버영재교육원 C언어 레벨 1 과정을 수강했을 때 만나서 친해져서 캠프까지 같이 갔었다. 

그때 숙소를 친한 친구들끼리 정할 수 있어서 현준, 다오, 준호(필자)의 이름 앞글자를 하나씩 합치니 방이름이 '현준다방'이 되었었는데, 

이때 멤버 그대로기 때문에 정한 이름이다. 누가 지었는지 몰라도 정말 재치있어요!



(MISC) Introduce

dlrpqkfhvmfform

 플래그 입력을 알려주는 문제로, ubuntu{} 사이에 있는 Flag를 입력하면 된다.


(MISC) OpenMe

3ncrypt3d_zip_fil3_i5_fun_t0_s0lv3

ZIP 파일 패스워드 크래킹 툴이 Advanced ZIP Password Recovery를 사용한 무차별대입공격(Brute-force)으로 

주어진 ZIP file의 압출을 하나씩 풀어나가면 된다. 

주어진 압축파일을 풀면 i, we, you 세 가지 폴더가 나오고 그 아래로 또 여러개의 압축파일이 있는데 

팀원당 하나씩의 폴더를 맡아 하위 파일들을 열면서 플래그를 찾기로 했다. 

플래그는 내가 탐색한 we/love/food/samgyeobsal 폴더 안에 있었다.

각 압축파일의 패스워드는 4~5자리의 숫자로 구성되어 있었다.

부지런히 [Save password to file]을 누르면서 패스워드까지 저장해두면서 하고

다른 팀원들에게도 하라고 했는데 진짜 했는지는 모르겠다.

혹시라도 모르는 마음이였는데 결국 쓸 일 없었다.

그래도 여기에라도 뻘짓하는 겸 치고 올려보자는 마음에서 정리해봤다.


위와 같이 we/love/food/samgyeobsal 폴더에서 플래그가 나왔다(people의 하위 폴더는 까볼 필요가 없어서 안 까봤다). 


아무튼 다같이 함께할 수 있던 문제라 좋았다. 꺄르르 꺌꺌




(Web) Js Counter

Js_Is_s1mpl3!!!

와아아 내가 푼 문제다! 근데 때려맞춘 느낌이 강해서 그냥 조용히 있어야 할... 흐앵

먼저 주어진 링크로 가보니 아래 같은 웹페이지가 나타난다(아래 카운트 수는 무시해주세용).

페이지 소스를 확인해보니, func.js 파일에서 난독화된 자바스크립트 코드가 있었다. 

자바스크립트 난독화 해제 사이트에서 난독화를 해제하니 아래 같은 코드가 나온다.

팀원 모두 웹해킹에 대한 지식이 전무한지라 다들 117번 누르고! 98번 누르고! 그러는 줄 알았다. 음?

그렇게 다같이 매크로로 광클을 시도했는데 아무 반응 없었다(그렇다. 바로 저 사진이 103648번 누르느라 고생한 현준이의 스샷).

헤메다가 코드를 자세히 보니까 flag 배열에 117, 98, 19, 110값을 넣고 flag_back 배열의 모든 원소를 ^11 연산해서 flag배열에 집어넣는 것 같았다. 

그렇게 대충 Codecademy에서 자바스크립트 강좌를 훑어봤던 기억이 있던 나는 Codecademy의 컴파일러로 즉석에서 짠 코드를 실행했다.

왠지 쪽팔려서 가렸다 그랬더니 저렇게 나왔다.
