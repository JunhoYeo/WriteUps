# NYPC 2018 - Day 0

## #1 [연습문제] 78 89 80 67
점수 : **100** / 100

입력받은 숫자를 Python `chr()`를 이용해서 대문자로 바꿔서 출력한다.

- 그냥 `input().split()`으로 리스트에 저장하면 되므로 N까지 저장할 필요는 없다(그냥 `input()`으로 넘기자).
- Python3에서는 `*`를 이용해서 배열의 내용을 공백으로 분리해서 출력할 수 있다(아래 예시).

```py
list = [1, 2, 3]
print(*list)
# 1 2 3
```

- 준서랑 숏코딩 내기를 해서 제출횟수 1회 날리고 한줄로 만들어서 이겼다.
